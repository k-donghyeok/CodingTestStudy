# CodingTestStudy 🧠💻

코딩 테스트 대비를 위해  
다양한 알고리즘 문제를 직접 풀고 정리한 저장소입니다.

## 📌 목적
- 자료구조 / 알고리즘 실전 감각 유지
- 문제 해결 접근 방식 기록
- 코딩 테스트 및 기술 면접 대비

## 🛠 사용 언어
- Python

## 📂 문제 출처
- 이것이 취업을 위한 코딩테스트다 with 파이썬

 

## ✍️ 풀이 방식
- 가능한 한 **가독성 있는 코드** 작성
- 문제마다 **시간 복잡도 고려**
- 필요 시 주석으로 접근 방식 설명

## 📈 진행 현황
- 꾸준히 업데이트 중

> 단순 정답 코드 저장이 아니라  
> **문제 해결 과정을 기록하는 것을 목표로 합니다.**
